# SHOOTING_GAME

宇宙船を操作しながら弾を発射して隕石や敵を倒していくシンプルなシューティングゲームです。

[ここからプレイできます](https://0otk.github.io/SHOOTING_GAME/)

### 操作方法

・Spaceキー　ゲームスタート  
・Fキー　弾の発射  
・左右キー　左右の移動  

タイトル画面左上のマークをクリックすると上位5つのスコアが表示されます。  
※上記URLからプレイする場合ゲームを閉じると記録がリセットされます。

### 使用言語・ライブラリ
Python / Pyxel

### 開発経緯
大学のゼミで課されたPythonでの個人制作に際して作成しました。  
ゲームデザインや仕様、制作計画まですべて自分で設計して取り組みました。

### 開発の工夫
クラスを使って自機の操作や敵の挙動、描画などをそれぞれ分離して管理できるようにコードを組みました。  
また、テキストファイルの読み込み・上書きを通じて疑似的なロード・セーブ機能を実装しました。  

### 苦労した点・頑張った点
AtCoderやpaizaで行ってきたコーディングの知識を算術的問題以外の場面で応用するのに苦労しましたが、  
実装したい処理を論理立てて設計し、そのためにどの知識が使えるのかを考えることを頑張りました。  
この経験を通じて、今までインプットしてきたPythonの知識をアウトプットすることができました。  
また、その逆で今までの経験で触れることのなかった関数やモジュールについても開発を通じて多数学ぶことができ、  
特にクラスやos、sysやテキストファイルの読み込みといった知識は今後の開発にも活かせそうだと感じながら学べました。
